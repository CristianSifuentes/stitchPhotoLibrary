# DaVicio Photo Library (React 2026-style)

An advanced, feature-sliced React + TypeScript architecture inspired by modern 2025/2026 React practices:

- Feature-first boundaries (`features/photoLibrary/*`)
- Layer separation (`services`, `store`, `hooks`, `views`, `types`)
- App shell orchestration in `app/views`
- CSS Grid + Flexbox hybrid layout with responsive adaptation
- Modern React UX primitives (`useTransition`, `useDeferredValue`, memoized selectors)

## Run

```bash
npm install
npm run dev
```

> If your environment blocks npm registry access, install dependencies in a network-enabled environment.
